# Scroll Reveal UI 🚀

A modern dark landing page concept focused on smooth scroll animations and a top progress indicator.

This project demonstrates how small UI details like motion and feedback can significantly improve the user experience.

---

## ✨ Features

* 📊 Scroll progress bar (top indicator)
* 🎬 Reveal animations on scroll (Intersection Observer)
* 🌑 Modern dark UI design
* 🎯 Clean layout with reusable components
* ⚡ Smooth transitions and subtle motion effects

---

## 🛠️ Built With

* HTML5
* CSS3 (custom properties, modern layout)
* JavaScript (Intersection Observer API)
